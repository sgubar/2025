Варіант 2 (колір графіку - cиній; колір координатних осей та поділок - білий; колір фону - зелений)

1.
main(int argc, char* argv[]): головна функція та точка входу до програми.
display(): Основна функція малювання.
keyboard(unsigned char key, int x, int y): Функція зворотного виклику (callback), яка реагує на натискання звичайних клавіш (наприклад, 'r', '+', '-').
specialKeys(int key, int x, int y): Функція зворотного виклику для спеціальних клавіш (стрілки).

2.
glutInit(&argc, argv): Ініціалізує бібліотеку GLUT.
glutInitWindowSize(700, 700): Встановлює початковий розмір вікна в пікселях (ширина та висота).
glutInitWindowPosition(50, 50): Встановлює початкову позицію верхнього лівого кута вікна на екрані.
glutCreateWindow("Interactive Parametric Plot"): Створює вікно з заданим заголовком.
glutDisplayFunc(display): Реєструє функцію (display), яка буде викликатись для перемалювання вікна.
glutKeyboardFunc(keyboard) та glutSpecialFunc(specialKeys): Реєструють функції-обробники подій від клавіатури.
glutMainLoop(): Запускає головний цикл обробки подій. Програма входить у цей цикл і очікує на події (натискання клавіш, рух миші, запити на перемалювання), викликаючи відповідні зареєстровані функції.

3.
glBegin(mode) та glEnd(): Ці функції працюють у парі та позначають початок і кінець опису вершин для створення графічного примітиву.
Параметр mode у glBegin() визначає, який саме примітив буде створено з переданих вершин. У коді використовувалися:
GL_LINES: Малює набір окремих відрізків. Кожні дві вершини, описані між glBegin та glEnd, утворюють одну лінію. Використовувався для малювання осей та поділок.
GL_LINE_STRIP: Малює послідовність з'єднаних відрізків. Кожна нова вершина з'єднується з попередньою. Використовувався для побудови плавної кривої графіка.

4.
glClearColor(r, g, b, a): Ця функція встановлює колір, який буде використовуватися для очищення екрана.
Параметри: r, g, b — інтенсивність червоного, зеленого та синього кольорів (0.0-1.0), альфа-канал (прозорість).
glClear(GL_COLOR_BUFFER_BIT): Ця функція виконує очищення. Вона заповнює буфер кольору тим значенням, яке було попередньо встановлено функцією glClearColor.
Параметр GL_COLOR_BUFFER_BIT — це константа, яка вказує, що потрібно очистити саме буфер, який відповідає за колір зображення.

5.
GL_POINTS: Малює окремі точки (вершини).
GL_TRIANGLES: Малює набір окремих трикутників (кожні 3 вершини утворюють один трикутник).
GL_QUADS: Малює набір окремих чотирикутників (кожні 4 вершини).
GL_POLYGON: Малює єдиний опуклий полігон (багатокутник).
